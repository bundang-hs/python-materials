🌱이 곳에는 수업자료(pdf)와 예시코드가 업로드 됩니다

1. 수업자료 파일을 참고하세요. 수업시간에 앞에 있는 칠판을 보고 코드를 입력하기 힘들테니!
2. 선생님이 올려준 예시파일에서 코드를 복사하여 파이참에 붙여넣기 하세요.
3. 코드를 수정하여 저장한 후 다시 업로드하세요.
   [ STEP 1 ] 내 저장소 만들기

   선생님의 링크를 타고 들어가 내 이름(학번)을 찾으세요.
 
   "Creating repository..." 메시지가 끝나면 내 이름이 붙은 저장소 링크가 생깁니다. 클릭!

   [ STEP 2 ] 파이참(PyCharm) 활용

   깃허브의 예시 코드를 복사해서 파이참에 붙여넣으세요.

   print('안녕') 같은 과제 내용을 채우고 실행(Run)해 봅니다.

   [ STEP 3 ] 깃허브에 업로드 (제출)

   완성된 코드를 다시 복사해서 깃허브의 해당 파일로 갑니다.

   ✏️ 연필 아이콘(Edit) 클릭 → 내용 붙여넣기 → 🟢 Commit changes 클릭!


   ⚠️ 주의사항 (필독!)
   이메일 인증 필수: 깃허브 가입 후 메일함에서 인증을 안 하면 저장이 안 됩니다!

   파일명 유지: 선생님이 처음에 준 파일 이름을 절대 바꾸지 마세요. (예: main.py 그대로 유지)

   제출 확인: 본인 저장소 메인 화면에 내가 수정한 코드가 보이면 제출 성공!
